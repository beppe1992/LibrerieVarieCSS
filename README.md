# LibrerieVarieCSS
In questo progetto sono riportate alcune librerie generiche CSS, installate via NPM.<br>

## CSSPin
Libreria di Spinner CSS, per provarla è necessario lanciare il comando http-server (se non è installato, installarlo con il comando npm install http-server -g) e collegarsi al link http://localhost:8080/app/csspin/Esempio.html. <br>
