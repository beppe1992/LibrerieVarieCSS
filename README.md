# EsempiLibrerieCSS
In questo progetto sono riportate alcune librerie generiche CSS, installate via NPM.<br>

## CSSPin
Libreria di Spinner CSS, per provarla è necessario lanciare il comando http-server (se non è installato, installarlo con il comando npm install http-server -g) e collegarsi al link http://localhost:8080/app/csspin/Esempio.html. <br>


# EsempiLibrerieJS
In questo progetto sono riportate alcune librerie generiche JS, installate via NPM.<br>

## Offline.js
Libreria che si occupa di verificare la corretta connessione alla rete, per provarla è necessario lanciare il comando http-server (se non è installato, installarlo con il comando npm install http-server -g), collegarsi al link http://localhost:8080/app/offline-js/Esempio.html e disabilitare la connessione di rete (con Chrome è possibile farlo attraverso gli strumenti per sviluppatori). <br>
